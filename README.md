# 🌐 Мои соцсети

Минималистичная страница-визитка со ссылками на все мои профили в социальных сетях.  
Адаптивный дизайн, тёмная тема, плавные анимации.

![Превью страницы](https://via.placeholder.com/800x400/1a1f2f/ffffff?text=Мои+соцсети)

## 🚀 Быстрый старт

1. Скачайте файл `index.html`
2. Откройте его в любом браузере
3. Готово! Страница уже работает

## 📱 Включенные соцсети

- **Telegram** — @yourchannel
- **Rossi** — ваш профиль
- **YouTube** — @yourchannel
- **TikTok** — @yourhandle
- **Twitch** — ваш канал

## ⚙️ Настройка

### 1. Изменить ссылки
В самом низу файла `index.html` найдите блок `<script>` и замените ссылки-заглушки на свои:

```javascript
const links = {
  telegram: 'https://t.me/ваш_канал',
  rossi: 'https://rossi.example.com/ваш_профиль',
  youtube: 'https://youtube.com/@ваш_канал',
  tiktok: 'https://tiktok.com/@ваш_логин',
  twitch: 'https://twitch.tv/ваш_канал'
};
